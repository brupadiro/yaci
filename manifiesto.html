Manifiesto

Esta plataforma no persigue neutralidad visual. Construimos una interfaz con gravedad, silencio y foco táctico: cada decisión sirve al clima narrativo.

Oscuridad útil

Partimos de un principio: la estética debe resolver intención. El negro profundo no está para dramatizar gratis, sino para aislar señal, concentrar atención y reducir ruido cognitivo.

El verde desaturado funciona como continuidad de lectura. El acento brillante aparece poco, solo cuando una acción necesita prioridad inequívoca.

Evitamos superficies limpias y perfectas. El grano, las líneas y las variaciones tonales recuerdan que toda interfaz es un medio, no un objeto terminado.

1. Menos brillo, más jerarquía.

2. Menos ornamento, más atmósfera.

3. Menos novedad, más coherencia interna.

4. Menos tendencia, más identidad operativa.
